Multiple Storefronts Mod for our keys selling platform.

Mod allows to run several storefronts (or domains) with unique designs from the same installation. Every storefront have separate set of users, products, menus, CMS pages and URLs so they look totally different.

In terms of PC CD keys selling there is an ability to use general or only own stock for the domain

Split back-end mod allows to set up per-domain access to certain staff members, making certain domains hidden for them from management/stats.
