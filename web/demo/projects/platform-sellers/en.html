Sellers Module Mod for our keys selling platform.

Module which allows users to upload and sell CD keys on the site. We can even call it "Marketplace", as customers can rate sellers after purchase, helping other customers with decision.

Flexible commission level settings per seller. Clean and easy to understand dashboard and stats. Easy UI for keys upload.

New sellers require to set up bank accounts and upload ID docs. Every action can be moderated for new/untrusted sellers.

Money from sales added to user wallet with further in-app payouts to PayPal / Skrill / Bank wire accounts.
