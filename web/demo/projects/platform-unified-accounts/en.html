Unified accounts Mod for our keys selling platform.

Before that change users had to create a separate user account per user role (one account for customer, one for partner, one for seller). It was a lack of the engine we built our platform on.

Major rework has been done and now it allows to have different roles for the same user account. Furthermore user wallet is now used for all in-app operations (buying, selling, getting commissions, refund via wallet etc.)

We finally got rid of username as unique and required field which was used as user ID. Now it replaced with email.
